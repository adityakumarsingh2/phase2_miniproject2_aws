This project is a simple static website created with HTML and CSS, prepared to be uploaded to the aditya-miniproject2-s3 Amazon S3 bucket and shared through a public website URL.

Project Overview

The goal of this assignment is to create a basic webpage, upload it to the aditya-miniproject2-s3 Amazon S3 bucket, enable static website hosting, and make the website publicly accessible. This page explains the task, the hosting process, and the final result in a clean presentation.

Website

A responsive HTML page with simple styling, clear content, and a professional layout.

Hosting

Amazon S3 is used as the storage and hosting service for the static website files.

Access

The bucket policy allows public read access so visitors can open the website from a browser.

Task Workflow

To complete the assignment, the website file is created first and then configured in AWS S3 for public static hosting.

  1. Create an index.html file for the website homepage.
  2. Create a unique S3 bucket from the AWS Management Console.
  3. Upload the index.html file into the S3 bucket.
  4. Enable static website hosting and set index.html as the index document.
  5. Add a bucket policy that allows public read access for website visitors.

Final Output

After the file is uploaded and made public, the project can be opened using the S3 public object URL.

https://aditya-miniproject2-s3.s3.ap-southeast-2.amazonaws.com/index.html
